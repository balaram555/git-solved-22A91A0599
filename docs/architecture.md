# System Architecture

## Overview
DevOps Simulator follows a stable microservices architecture suitable for production deployments.

---

## Production Architecture
- Node.js application server
- PostgreSQL database
- Secure communication
- Monitoring and logging enabled

---

## Experimental Architecture
⚠️ Experimental architecture exists for research purposes only and is **disabled by default**.

Planned experiments include:
- AI-based monitoring
- Multi-cloud orchestration
- Chaos engineering
